# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-12-23

### Added
- Initial release
- Tournament CRUD operations (Create, Read, Update, Delete)
- Support for 4 tournament types: Futsal, E-Sport, Catur, Custom
- Participant management with auto-seeding
- Single-elimination bracket generation
- Real-time score updates with automatic winner advancement
- Champion celebration page with animations
- Admin password protection for tournaments
- View-only mode for spectators
- Responsive design for mobile and desktop
- LocalStorage data persistence
